## Configure the project structure:

**This is the structure of the project:** 👌

```plaintext
/
├── demo
├── dist
├── node_modules
├── public
├── src
│   ├── fonts
│   ├── img
│   ├── js
│   ├── scss
├── .gitignore
├── index.html
├── LICENSE
├── package-lock.json
├── package.json
├── postcss.config.cjs
├── README.md
```

